*Prerequisite*
1.selenium
To install: pip install selenium OR pip3 install selenium

2.Chrome Driver
https://sites.google.com/a/chromium.org/chromedriver/downloads


*How to use*
1.Install selenium
2.Download Browser Driver(depending on version) and place the downloaded driver in Driver Folder
3.Open CMD use command: python instagram_follower_followering_bot.py OR python3 instagram_follower_followering_bot.py
4.Input accordingly, ONLY USE instagram handler and NOT email
